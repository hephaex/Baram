# [주니어전자]톡톡 영어 - “잘못 배달된 것 같아요”

**전자신문** | 2025-12-30 00:00 | 

---

대화 (Conversation)Customer: Excuse me, I think this was delivered to the wrong address.(저기요, 이게 잘못 배달된 것 같아요.)Delivery Person: Oh, really? Let me check.(아, 정말요? 확인해볼게요.)Customer: Yes, this isn&#x27;t my order.(네, 이건 제 주문이 아니에요.)Delivery Person: I&#x27;m sorry about that. I&#x27;ll take it back and fix it right away.(죄송합니다. 가져가서 바로 처리하겠습니다.)유용한 표현들 (Useful Expressions)It was delivered to the wrong address. → 잘못된 주소로 배달됐어요.I think this is not my order. → 이건 제 주문이 아닌 것 같아요.I&#x27;ll fix it right away. → 바로 처리하겠습니다.Can you check it for me? → 확인해 주실 수 있나요?
