# “해킹보다 내부자 위협이 문제”…폴라리스오피스, 협업 보안 솔루션 출시

**디지털데일리** | 2026-01-06 00:00 | 

---

[디지털데일리 이안나기자] 폴라리스오피스는 아틀라시안의 보안 솔루션 ‘아틀라시안 가드 프리미엄’을 국내 시장에 정식 출시했다고 6일 밝혔다.아틀라시안 가드 프리미엄은 협업 환경에서 발생하는 데이터 접근과 사용자 활동을 관리·통제하기 위한 엔터프라이즈 보안 솔루션이다. 아틀라시안은 ‘지라(Jira)’와 ‘컴플루언스(Confluence)’ 등 협업 도구를 제공하는 글로벌 기업으로, 국내에서도 대기업과 IT 기업을 중심으로 활용되고 있다.이번 출시는 기업 보안 위협이 외부 침입뿐 아니라 내부 권한 오남용으로 확대되는 흐름에 맞춰 이뤄졌다. 특히 중대 정보 유출 사고에 대한 규제 강화가 예고되면서 내부 데이터 접근을 관리하는 통제 체계에 대한 기업들의 관심이 높아지고 있다.아틀라시안 가드 프리미엄은 기존 싱글사인온(SSO) 기반 접근 제어를 넘어 데이터 분류, 사용자 활동 모니터링, 콘텐츠 스캐닝 기능을 포함한다. 협업 도구 내 데이터 흐름을 지속적으로 점검해 이상 접근이나 민감 정보 노출 가능성을 식별하는 방식이다.예를 들어 특정 사용자가 단기간에 대량의 문서를 다운로드하거나, 프로젝트 문서에 개인정보·결제 정보 등 민감 정보가 입력될 경우 이를 감지해 관리자가 확인할 수 있도록 지원한다. 사고 발생 이후 대응보다는 사전 탐지와 관리에 초점을 맞춘 구조다.폴라리스오피스 측은 “내부 직원의 데이터 접근 과정에서 발생할 수 있는 보안 리스크에 대한 관리 요구가 커지고 있다”며 “기존 보안 체계로는 파악하기 어려운 협업 환경 내 사용자 활동을 가시화해, 기업이 규제 환경 변화에 대응할 수 있도록 지원할 계획”이라고 말했다. 이어 “단순한 솔루션 공급을 넘어, 기업의 데이터 관리 체계를 지원하는 역할을 하겠다”고 덧붙였다.한편 폴라리스오피스는 아틀라시안의 ‘플래티넘 솔루션 파트너’로, 이번 제품 출시를 계기로 기업용 협업·보안 솔루션 사업을 확대한다는 방침이다.
