# 간 축적 지방 직접 찾아가 제거한다...KIST, 지방간 新치료제 개발

**전자신문** | 2025-12-30 00:00 | 

---

비만과 대사증후군 확산으로 &#x27;대사이상 지방간질환(MASLD)&#x27; 환자가 빠르게 늘어나는 가운데, 우리 연구진이 질환 원인을 직접 해결하는 새로운 치료법을 개발했다.한국과학기술연구원(KIST·원장 오상록)은 이현범·박진영 생체분자인식연구센터 박사팀이 이준석·전대원 한양대 교수팀과 간세포 내 지방을 직접 제거하는 새로운 치료제를 개발했다고 30일 밝혔다.기존 치료는 주로 식이 조절이나 운동, 약물로 지방 대사 과정을 간접 조절하는 방식으로 이미 간세포 내에 쌓인 지방을 제거하는 데에는 한계가 있었다. 반면 개발 치료제는 지방을 표적으로 인식해 붙잡아 분해하는 방식이다. 원인 물질 자체를 제거한다.

연구팀은 간세포 안에 축적된 지방방울을 직접 줄이기 위해, 지방 인식 물질과 지방 분해 효소를 하나로 결합한 치료제(LDI)를 설계했다. 이는 간에 이미 쌓인 지방에 직접 작용하도록 만들었다.개발 치료제는 지방간 유도 세포와 동물모델 실험으로 효과·안전성이 검증됐다. 간에 축적된 지방과 염증 반응이 뚜렷하게 감소했으며 간 손상 지표도 최대 84%까지 개선됐다. 또 독성이 관찰되지 않아 실제 치료제 개발 가능성을 확인했다.이번 성과는 아직 승인된 치료제가 거의 없는 지방간 질환 분야에서 새로운 치료 플랫폼이 될 가능성을 보여준다. 지방을 직접 표적하는 차별화된 방식으로 신약 개발에 유망하고, 빠르게 성장하는 글로벌 지방간 치료제 시장에서 경쟁력을 가질 수 있다. 나노기술 기반 구조를 활용, 다른 대사질환 치료제로도 확장할 수 있다.이현범 KIST 박사는 “그동안 지방간 치료 방법이 제한적이었지만, 이번 연구로 새로운 간 건강 회복 방향을 제시했다”며 “앞으로 실제 환자에게 적용할 수 있도록 치료제 개발을 계속해 더 많은 사람들이 건강한 삶을 누리는 데 기여하고 싶다”고 말했다.이번 연구 성과는 국제 학술지 어드밴스드 머터리얼즈 표지논문으로 선정됐다.
