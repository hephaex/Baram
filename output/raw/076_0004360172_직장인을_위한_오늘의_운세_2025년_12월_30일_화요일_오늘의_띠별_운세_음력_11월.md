# [직장인을 위한 오늘의 운세] 2025년 12월 30일 화요일 오늘의 띠별 운세 (음력 11월 11일)

쥐띠 : 꾀꼬리가 버들가지 위에 깃을 치니 가지 마십시요. 조각 조각이 황금입니다.60년생 : 구설수를 조심하십시요.72년생 : 때를 만났으니 이름을 떨치고 가정에 경사가 있어 기분이 좋을 것입니다.84년생 : 그 동안 마음 고생이 심해 자칫 긴장이 풀려 몸을 상할 수 있으니 주의하십시요.96년생 : 돌을 쪼아 옥을 보니 힘써 노력하면 반드시 얻는 바가 있을것입니다.소띠 : 모든 일이 어려울 듯하면서도 좋은 방향으로 간다.61년생 : 시기적으로 아주 좋은 운을 타고 있으니 뜻한 대로 크게 성공할 수가 있습니다.73년생 : 육체적인 건강, 정신적인 건강은 거의 함께 합니다.85년생 : 산이고 강이고 원하는 곳으로 떠나십시요. 함께 하고 싶은 사람이 있다면 같이 떠나도 좋습니다.97년생 : 목적하는 것이 있다면 뜻대로 밀고 나가십시요.범띠 : 만인이 공로를 치하하며 받들게 됩니다.62년생 : 포기하고 싶었던 일들이 하루가 멀다 하고 빠르게 진척이 있습니다.74년생 : 언론 상에 영웅으로 추대 받게 되는 공을 세우니 한낮 삶의 역경이라 생각하고 두려워 마십시요.86년생 : 각종 분야에 노력을 닦아온 결실이 맺어질 때입니다.98년생 : 사업이 번창하게 됩니다.토끼띠 : 기존의 나쁜 습관이나 관습을 버리고 새로운 삶을 계획합니다.63년생 : 눈앞의 이익을 생각하면 사업은 할 수가 없습니다. 먼 훗날을 기약하고 장래를 생각해 투자하는 것이 바람직하다75년생 : 자꾸 방향을 바꾸고 선택을 다시 하게 되면 안 좋은 결과를 가져올 수 있습니다.87년생 : 새로운 자세로 일에 임하게 됩니다.99년생 : 지금 당장 이루어지는 것은 아무 것도 없습니다. 조급해 말고 때를 기다려 보십시요.용띠 : 하늘의 뜻이 곳 나의 뜻이니 어찌 이루지 못하리요.00년생 : 어려운 시기가 지나 이제야 호기를 만나니 가지고 있던 기량을 마음껏 발휘한다면 성공하게 될 것입니다.64년생 : 환자의 정신이 많이 지쳐 있어 병세가 악화되니 일단 충분한 휴식이 필요합니다. 중증입니다.76년생 : 시기하는 이가 많아 어려움을 겪겠으나 포기하지말고 끝까지 나가면 끝내는 이루게 될것입니다.88년생 : 너무 높은 목표를 세웠으니 올라가는데 힘이 듭니다.뱀띠 : 목표를 향해 빠르게 움직이면 큰 소원도 능히 성취할것입니다.01년생 : 시간을 낭비하면 안 됩니다. 좋다면 무조건 밀고 나가십시요.65년생 : 여행을 떠나십시오. 짝이 없는 귀하라면 여행도중 꿈속에 그리던 상대를 만나게 됩니다.77년생 : 조금 더 부지런히 움직이고, 매사에 과감한 추진력이 필요합니다.89년생 : 생각도 못했던 일로 즐거워집니다.말띠 : 외지로 나가지 마십시요.02년생 : 이익도 없고 고생만 하게 되니 소원을 이루기 어렵겠습니다.66년생 : 생각지도 않았던 불상사가 생길 수 있겠으니 마음의 안정을 취하십시오.78년생 : 남과 다투지 마십시요. 몸을 다칠 수가 있습니다.90년생 : 모든 물건에는 각각 주인이 있으니 남의 물건을 탐내지 마십시요. 망신을 당할것입니다.양띠 : 스스로를 위로하려 하지만 그럴수록 마음이 혼란스럽기만 합니다.55년생 : 사업에 실적이 늘어나질 않아 매출은 줄어들고 사세를 좁혀야 하는 지경에까지 이르게 됩니다.67년생 : 바라는 꿈이 너무 허황된 것 같습니다.79년생 : 어디를 가도 반가워하는 사람도 없으니 집안에 머무는 것이 좋겠습니다.91년생 : 흑백을 가리기는 힘듭니다. 작당한 선에서 타협하십시요.원숭이띠 : 깊은 산중에 길을 잃고 헤매다가 호랑이를 만나니 어려움이 극에 달했습니다.56년생 : 서쪽으로 가면 길합니다.68년생 : 길 밖으로 나가지 마십시요. 낭패만 당하고 사시 돌아올 것입니다.80년생 : 마음이 작고 담이 크니 항상 안정을 누리십시오.92년생 : 혼자의 힘으로 목표 달성은 어렵겠습니다.닭띠 : 계획성 없이 일을 해 나가니 모든 것이 엉망입니다.57년생 : 어느 것도 지금은 뜻대로 되는 것이 없습니다. 이런 때에 사업을 확장하면 크게 실패할 수 있습니다.69년생 : 병이 들면 위독한 상태까지 가게되니 주의하십시요.81년생 : 여행은 떠나지 마십시요. 지금 시기가 좋지 않습니다. 다음으로 연기하십시요.93년생 : 모든 일을 쉽게 이루니 기쁨이 가득합니다.개띠 : 자신의 의지를 확고히 할 때입니다.58년생 : 전업이나 개업을 하려 한다면 확실한 결단을 내려라 주위의 유혹을 이겨야 가능합니다.70년생 : 정신적인 스트레스에 시달리게 됩니다.82년생 : 꼭 소송을 해서 시비를 가려야 한다고 생각한다면 과감하게 밀고 나가십시요.94년생 : 못했던 공부에 집중하면 좋은 결과가 있을 것입니다.돼지띠 : 단시일 내에 이루려는 꿈은 불길합니다.59년생 : 무리한 투자, 눈앞에 보이는 이윤에 뛰어들려 하지 마십시요.71년생 : 나이 들어 응시한 시험에 합격하는 경우가 많습니다.83년생 : 시야를 넓게 가지고, 천천히 단계를 밝아 올라가도록 하십시요. 반드시 꿈이 이루어집니다.95년생 : 허욕이나 욕심을 버리고 직책에 충실하여야 나중에 좋을 것입니다.
