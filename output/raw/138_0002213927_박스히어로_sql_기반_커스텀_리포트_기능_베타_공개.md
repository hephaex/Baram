# 박스히어로, SQL 기반 커스텀 리포트 기능 베타 공개

**디지털데일리** | 2025-12-30 00:00 | 

---

[디지털데일리 이상일기자] 비지피웍스(대표 문희홍)는 자사 재고관리 솔루션 ‘박스히어로’에 사용자 맞춤형 커스텀 리포트 기능을 베타 버전으로 선보였다고 밝혔다.재고를 관리하는 기준은 업종이나 취급 품목에 따라 상이하다. 어떤 사업자는 유통기한을 핵심 지표로 삼는 반면, 다른 사업자는 재고 회전율이나 소진 속도를 중심으로 관리한다. 하지만 기존 재고관리 시스템에서 제공하는 리포트는 대부분 정형화된 형태에 머물러 있어, 실제 업무 현장에서 필요한 정보와 괴리가 발생하는 경우가 많았다.회사 측은 박스히어로의 커스텀 리포트 기능 역시 이러한 문제 인식에서 출발했다고 설명했다. 다양한 업종의 고객이 서비스를 이용하는 과정에서 “데이터는 축적되지만 원하는 형태로 확인하기 어렵다”는 의견이 지속적으로 제기됐다는 것이다.이에 박스히어로는 특정 업종이나 제한된 활용 사례에 맞춘 리포트를 추가하는 방식 대신, 사용자가 직접 원하는 기준에 따라 리포트를 구성할 수 있는 커스텀 리포트 기능을 도입했다. 재고 수량과 입출고 이력 등 기본 데이터는 동일하지만, 이를 어떤 기준으로 분류하고 분석할지는 사용자마다 다를 수 있다는 판단에서다. 예를 들어 특정 거래처를 제외한 출고량을 확인하거나, 최근 몇 개월간 회전율이 낮은 품목만 선별하는 등 상황에 맞춘 데이터 조합이 가능하다.베타로 제공되는 커스텀 리포트 기능은 SQL 기반으로 운영된다. 외형상 개발자 도구처럼 보일 수 있으나, 회사 측은 이는 모든 사용자의 요구를 단일한 리포트 형식에 맞추기보다 각자의 관리 기준을 최대한 반영하기 위한 선택이라고 설명했다. 아울러 최근에는 챗GPT 등 생성형 AI를 활용해 SQL에 익숙하지 않은 사용자도 비교적 수월하게 원하는 리포트를 설계할 수 있는 환경이 마련되고 있다는 점도 고려했다.이번 커스텀 리포트 기능 베타 출시는 재고 관리 데이터를 단순한 기록을 넘어 의사결정을 지원하는 자료로 활용 범위를 확장하려는 시도의 일환이다. 박스히어로는 베타 운영 기간 동안 사용자 피드백을 수렴해 활용성을 점검하고, 향후 접근성을 높이는 방향으로 기능을 개선해 나갈 계획이다.
